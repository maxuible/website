Deciding the VPS

I like the idea of creating from "scratch" and simple HTML. Also I wanted cost to be low, as this is mostly a learning exercise. So I chose Oracle Cloud Free Tier; Here are the benefits:

  1. Free
  2. Start from "scratch"

I don't expect this site's traffic to require a lot of computing/bandwidth. In the future I might change hosting services, but with the experiences gained from setting up this website up changing will be easier.

Setting Up the VPS

The "Stack" used:

  • VPS Service : Oracle
  • Host OS : Ubuntu
  • Web Server : nginx
  • DNS Service : Google Domains
  • SSL Certification : Certbot/LetsEncrypt

This is the simplest way I could find to set up a website without using an all in one service like WordPress. I am also using git to pull changes from a repository.
